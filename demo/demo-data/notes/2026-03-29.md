## Debugging: Pagination Offset Bug

Users are reporting duplicate items on page transitions. Root cause: we're using **offset-based** pagination but rows get inserted between requests.

```sql
-- Current (broken for inserts between pages)
SELECT * FROM items ORDER BY created_at LIMIT 20 OFFSET 40;

-- Fix: cursor-based
SELECT * FROM items
WHERE created_at < $cursor
ORDER BY created_at DESC
LIMIT 20;
```
